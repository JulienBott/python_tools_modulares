
# python_tools_modulares

Este repositorio agrupa una serie de utilidades reutilizables en Python, orientadas a facilitar el desarrollo de aplicaciones de escritorio, herramientas técnicas y proyectos personales de forma limpia, modular y mantenible.

El objetivo es proporcionar **bases arquitectónicas** que puedan integrase en otros proyectos facilitando su utilización con pocas lineas de código y extenderse según las necesidades de cada proyecto.

Es un repositorio que ira creciendo con el tiempo a medida que necesite este tipo de tools en mis proyectos personales que ire también publicando en repositorios independientes en GitHub.

El repositorio se segmenta por carpetas cada una destinada a un ambito concreto. En cada una de estas carpetas se iran publicando distintas versiones (sin fecha prevista de implementación).

El repositorio viene acompañado de un contrato de __MIT Licence__ donde autorizo, sea a nivel personal o empresarial, el uso de __python_tools_modulares__ siempre y cuando se me reconozca su autoria original. Dicho contrato de MIT Licence tiene clausulas añadidas.


---

## Filosofía del repositorio

- arquitectura clara y desacoplada
- sistemas basados en herencias de clases propias
- código legible y documentado con ejemplos de uso
- reutilización sin dependencia rígida (mediante el uso de kwargs)
- sistemas hibridos entre atributos (y métodos) nativos y personalizados
- evolución incremental y controlada
